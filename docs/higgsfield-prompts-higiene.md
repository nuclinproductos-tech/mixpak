## Configuración técnica

| Parámetro | Valor |
|---|---|
| Herramienta | Higgsfield CLI v0.2.3 |
| Modelo | `flux_2` (FLUX.2 Pro) |
| Resolución | `1k` (1280 × 720 px) |
| Ratio | `16:9` |
| Coste | 1 crédito/imagen |
| Total créditos | 6 créditos (7 incluyendo test) |
| Formato guardado | `.jpg` (origen `.png` del CDN) |

## Para regenerar una imagen

```bash
# Sustituye PROMPT por el texto de la card deseada
higgsfield generate create flux_2 \
  --prompt "PROMPT" \
  --aspect_ratio "16:9" \
  --wait --json
# Descarga:
curl -sL "<result_url>" -o "img/sectores/higiene/<nombre-archivo>.jpg"
```
